# Aplicație Web pentru Programări la Salon

## Descriere
Această aplicație web permite gestionarea programărilor într-un salon de coafură / frizerie pentru bărbați, femei și copii.

Aplicația este realizată folosind tehnologii web de bază:
- HTML
- CSS
- JavaScript

Datele sunt salvate local folosind localStorage.

## Funcționalități
- Adăugare programare
- Selectare tip client (bărbat / femeie / copil)
- Afișare listă programări
- Salvare automată a datelor

## Tehnologii folosite
- HTML5
- CSS3
- JavaScript (ES6)
- Git & GitHub

## Rulare aplicație
1. Se descarcă proiectul
2. Se deschide fișierul `index.html` într-un browser
